# Ramda Talk

The deck for a talk about Ramda, written with [remark](https://github.com/gnab/remark)

## Presenting

Press `c` to open a closed, mirrored page.
Press `p` to enter presenter mode

## Development

```bash
yarn install
yarn deck:watch
```

## Deployment

Build with

```bash
yarn deck
```

Then push to github to deploy to gh-pages.
